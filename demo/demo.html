jQuery.tabSlideOut Demo

Left

This panel doesn't close automatically when somewhere else is clicked.

And it can be controlled from the buttons in the main window.

Note also the support for Font Awesome icons in the tab handles.

Right

This panel opens by default.

This tab will close when when another part of the page is clicked, but not if the checkbox below is checked and never when a button is clicked.


Top (hover)

This tab is set to bounce faster, longer, and less than the default values used by the other tabs.

It is also offset from the right side, and its tab is offset from the right too.

Bottom

This panel closes automatically when somewhere else is clicked, except another panel (default behaviour).

It has both an offset and an otherOffset, meaning its edges stay a constant distance from the viewport edges.

Finally, it asks for confirmation on close.
